## About This Project

This is my personal portfolio website at [ilia.to](https://ilia.to), built with Next.js 16 (App Router), React 19, TypeScript, and Tailwind CSS. The site serves as an interactive resume — it renders my full CV directly on the homepage from an HTML source, with email obfuscation for privacy and phone number linkification for mobile usability. A floating PDF download button provides quick access to the printable version. The project uses a modern CI/CD pipeline with GitHub Actions, Husky pre-commit hooks, and automated linting, typechecking, and testing to maintain code quality.
